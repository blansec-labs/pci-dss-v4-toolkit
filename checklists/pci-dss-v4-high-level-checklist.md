# PCI DSS v4 High-Level Checklist

## Requirement 1 — Network Security Controls
- Firewall rules documented
- Inbound/outbound traffic restricted
- Network segmentation reviewed

## Requirement 2 — Secure Configurations
- Hardening baselines defined
- Default passwords removed
- Secure configuration standards implemented

## Requirement 3 — Protect Stored Account Data
- Sensitive authentication data not stored
- PAN encrypted
- Retention policies documented

## Requirement 4 — Encrypt Transmission of Cardholder Data
- TLS configured securely
- Weak protocols disabled
- Certificate inventory maintained

## Requirement 5 — Protect Systems Against Malware
- Endpoint protection deployed
- Anti-malware monitoring active
- Signatures updated

## Requirement 6 — Secure Systems and Software
- Vulnerability management process defined
- Secure SDLC implemented
- Patch management active

## Requirement 7 — Restrict Access by Business Need
- RBAC implemented
- Access reviews documented
- Least privilege enforced

## Requirement 8 — Identify Users and Authenticate Access
- MFA enabled
- Unique IDs assigned
- Authentication policies documented

## Requirement 9 — Restrict Physical Access
- Physical access monitored
- Visitor logs maintained
- Media protection controls implemented

## Requirement 10 — Logging and Monitoring
- Audit logs enabled
- SIEM monitoring active
- Time synchronization configured

## Requirement 11 — Security Testing
- Vulnerability scans performed
- Penetration testing conducted
- IDS/IPS reviewed

## Requirement 12 — Security Policies and Programs
- Security policies approved
- Risk assessments performed
- Awareness program active
